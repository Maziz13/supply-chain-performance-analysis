
# Supply Chain Performance Analysis

## Overview
End-to-end analysis of DataCo Global Supply Chain dataset 
(180,000+ orders) to identify delivery bottlenecks, 
profitability drivers, and shipping efficiency gaps.

## Tools
- Python (Pandas) — Data Cleaning & Analysis
- Power BI — Interactive Dashboard (Coming Soon)

## Business Questions
- Q1: What is the late delivery rate across shipping modes and markets?
- Q2: Which markets and product categories generate the most profit?
- Q3: Which shipping mode is the most efficient?

## Key Findings
- 54.8% of orders are delivered late — a systemic global issue
- First Class has the highest late delivery rate (95%) despite being a premium service
- Computers category generates the highest profit per order (USD 157)
- Standard Class is the most reliable shipping mode (38% late delivery rate)
